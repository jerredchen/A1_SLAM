A1 SLAM: Quadruped SLAM using the A1's onboard sensors
======================================================

A1 SLAM is a rospackage that brings real-time SLAM capabilities utilizing factor graph optimization to Unitree's A1 quadruped.
This package was designed around the onboard sensors on the A1 and aims for easy, convenient installation.
In addition, this package is compatible with sensors that are not default to the A1.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
